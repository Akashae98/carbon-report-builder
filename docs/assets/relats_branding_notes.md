# Relats Branding Notes

## Purpose

This document summarizes the visual identity extracted from publicly available Relats materials and is intended to guide the design of generated PDF reports.

Important:

The Relats branding applies only to generated reports.

The application interface uses the Footprint Mappa brandbook.

---

# Brand Positioning

Relats presents itself as an industrial and engineering-focused company operating in sectors such as:

* Automotive
* Mobility
* Aerospace
* Renewable Energy
* Industrial Manufacturing

The visual language should communicate:

* Engineering
* Precision
* Reliability
* Technical expertise
* Sustainability
* Professionalism

Avoid styles that feel:

* Playful
* Startup-oriented
* Gaming-inspired
* Excessively futuristic

---

---

# Logo Usage

Assets location:

```txt
./logo_relats.png
```

Preferred usage:

* Cover page
* Header area
* Final page

Avoid excessive repetition.

The report should feel like a professional document prepared for Relats, not a marketing brochure.

---

# Typography Guidelines

If Relats fonts are unavailable:

Recommended fallback stack:

```css
font-family:
Inter,
Helvetica,
Arial,
sans-serif;
```

Style goals:

* Clean
* Readable
* Professional
* Print-friendly

Avoid decorative typography.

---

# Layout Principles

Reports should prioritize:

1. Readability
2. Data clarity
3. Professional presentation

Use generous whitespace.

Avoid crowded pages.

Use a clear hierarchy:

Heading
Subheading
Content
Chart
Conclusion

---

# Charts

Preferred style:

* Minimal
* High contrast
* Print-friendly

Suggested palette:

```css
#ff5710
#1c1c1c
#6b7280
#9ca3af
#d1d5db
```

Avoid:

* Neon colors
* Excessive gradients
* Heavy shadows

---

# Photography

If photography is used:

Preferred themes:

* Manufacturing
* Engineering
* Industrial processes
* Sustainability
* Materials

Images should support the report rather than dominate it.

* Strong typography

---

# Relationship with Footprint Mappa

The application and generated reports intentionally use different visual identities.

Application:

* Footprint Mappa
* SaaS experience
* Gradients
* Modern UI

Generated Report:

* Relats
* Corporate
* Industrial
* Print-oriented

This separation reflects a real-world SaaS platform generating client-branded reports.
