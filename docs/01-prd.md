# Product Requirements Document

## Vision
Transform carbon footprint datasets into professional PDF reports.

## MVP
Product Carbon Footprint (PCF)

Features:
- Upload CSV
- Validate data
- Generate metrics
- Dashboard
- Report preview
- PDF export

## Future Scope
- OCF support
- Multiple report templates
- AI summaries

## Success Criteria
A user can upload a CSV and export a professional report in minutes.

## Non Goals
- Carbon calculations
- Authentication
- Database persistence
- Admin panel
