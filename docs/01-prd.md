# Product Requirements Document

## Vision
Transform carbon footprint datasets into professional PDF reports.

## MVP
Product Carbon Footprint (PCF)

Features:
- Upload CSV
- Validate data
- Generate metrics
- Report preview
- PDF export
- Client brand preset selection
- Production persistence with Xano
- Configurable local persistence with filesystem (default) or Xano

## Future Scope
- OCF support
- Multiple report templates
- AI summaries
- Report history and comparison

## Success Criteria
A user can upload a CSV and export a professional report in minutes.

## Non Goals
- Carbon calculations
- Admin panel
- Report history UI
